# JC4880P443C_I_W_Remote v1.3.3

## Highlights

- Launcher home-screen page swipes now animate smoothly instead of switching pages instantly.
- The release keeps the live display rotation, recorder mic gain, and quick-access polish from the current 1.3.x line while packaging the latest validated firmware images.
- The repo also carries the latest joystick enclosure/CAD work that was already present in the workspace for this release cut.

## Validated Build Sizes

- Main ESP32-P4 OTA app image: `0x733090` bytes, leaving `0x08CF70` bytes free in either `0x7C0000` OTA slot.
- ESP32-C6 app image: `0x1C7E50` bytes, leaving `0x0181B0` bytes free in the 1920 KB app slot.
- ESP32-C6 merged flash image: `0x1D7E50` bytes.
